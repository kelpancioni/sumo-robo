# esp32-sumo-robot
Repositório para o projeto de robô autônomo de sumô na disciplina de Projeto em Sistemas Embarcados. O robô, utilizando ESP32 WROOM32 e programado em C/C++ com PlatformIO, deve empurrar o oponente para fora do ringue, respeitando limites de tamanho (10x10cm, 500g) e operando de forma autônoma com sensores e motores.
